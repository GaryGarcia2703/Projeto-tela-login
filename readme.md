# Projeto Tela de Login — Interface Estética

Este projeto consiste em uma interface estática inspirada em telas de login modernas.  
O foco principal do projeto é o design da página, a organização visual dos elementos e a simulação de uma interface real.

## Demo
Acesse o projeto através do link:  
https://garygarcia2703.github.io/Projeto-tela-login/

## Descrição do Projeto
A página apresenta um layout estilizado simulando uma tela de login, desenvolvida para treinar conceitos básicos de UI, alinhamento, tipografia e estética Front-end.

## Funcionalidades
- Interface visual simulando uma tela de login
- Estrutura simples composta por formulário estilizado
- Design responsivo básico

## Tecnologias Utilizadas
- HTML5  
- CSS3  

## Objetivo do Projeto
Praticar:
- Criação de interface visual
- Estruturação de formulários
- Estilização utilizando CSS com foco em identidade visual
